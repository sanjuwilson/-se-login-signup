# se-login-signup

A simple front-end web development project for Software Engineering homework.

## Project Description
This project contains two web pages:
- **Login page**
- **Signup page**

Both pages are built using only **HTML** and **CSS** and share the same stylesheet for consistent design.

## Files
- `login.html` - Login form page
- `signup.html` - Signup form page
- `styles.css` - Shared styling for both pages
- `README.md` - Project documentation

## Features
### Login Page
- User ID field
- Password field
- Submit button
- Link to Signup page

### Signup Page
- First Name field
- Last Name field
- User ID field
- Password field
- Submit button
- Link to Login page

## Styling
- Centered forms
- Consistent layout across both pages
- Clean spacing and alignment
- No external frameworks used

## How to Run
1. Download or clone the repository
2. Open the project folder in VS Code
3. Open `login.html` or `signup.html` in your browser
